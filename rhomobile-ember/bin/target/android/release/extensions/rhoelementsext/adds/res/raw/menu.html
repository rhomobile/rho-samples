Welcome to Motorola RhoElements. This is the default startpage. This page has been provided to get you and your application up and running with RhoElements. There is just one step required for first use:
  1. Change the default Config.xml
    /Android/data/
    com.motorolasolutions.rhoelements/
    Config.xml
The Config.xml sets default settings for all your RhoElements applications. To run your application:
  1. Open for editing:

    /Android/data/com.motorolasolutions.rhoelements/Config.xml

  2. Set the "value" parameter of the "StartPage" tag to the URL of your application's first page.
  3. Save Config.xml, and restart RhoElements.
